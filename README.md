# 🌐 New Website - Infinity

Desenvolvido pela equipe **Momple**, o **New Website - Infinity** tem como objetivo modernizar a presença digital da **Infinity School**, criando uma plataforma clara, responsiva e visualmente atrativa.

## 📖 Descrição do Projeto

O **New Website - Infinity** foi criado para destacar os cursos, valores e propósitos da instituição, oferecendo uma **experiência intuitiva e acessível** para alunos e visitantes.

### ✨ Principais Características:
- Design moderno e responsivo
- Animações interativas com GSAP
- Integração com backend robusto
- Foco em performance, escalabilidade e acessibilidade

## 💻 Como Rodar Localmente

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/seu-repo.git
```

2. Entre na pasta do projeto:

```bash
cd new-website-infinity
```

3. Instale as dependências:

```bash
npm install
```

4. Inicie o servidor local:

```bash
npm run dev
```

5. Acesse no navegador:

```
http://localhost:5173
```

## ⚙ Explicação Técnica

### 🔁 Fluxo do App

- O **frontend**, desenvolvido em **React**, é responsável pela renderização dinâmica das páginas e navegação com **React Router**.
- O **backend**, feito com **Node.js** e **PostgreSQL**, gerencia os dados e integra funcionalidades.
- Formulários usam o **Formspree** para envio simplificado.
- Animações são aplicadas com **GSAP**.
- Deploy do **frontend** via **Vercel** e do **backend** via **Render**.
